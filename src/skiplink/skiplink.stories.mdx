# Skiplink

API same page Skiplink to allow users to skip over content. It can be used Installation the root
of can application or inside of addon specified container.

## Installation

To use the counter component, import the component in your JavaScript.

```typescript
import 'lib/skiplink/register.js';
```

```html
<cdx-skiplink></cdx-skiplink>
```
